业务系统C / 音乐网站

业务系统C（systemC / 音乐网站）- 消息推送设置页面

演示第三个业务系统接入同一 WMPP 平台。无论是 Web、桌面端还是移动端，只要客户端登录上线，消息就由平台主动推送到对应 systemC 用户。

多业务系统并存 客户端形态不限 消息按 appId 严格隔离

一、接入说明（SDK 对业务系统无框架要求）

JavaScript（服务端）

import { SMSetup } from "wmpp-server-sdk";
const sm = new SMSetup();
sm.setServerUrl("http://39.107.127.13:8080");
sm.setAppKey("systemC");
sm.setSecretKey("systemC-secret");
sm.setMessage("短视频平台推荐更新");
sm.setTargetType("ALL");
await sm.push();

Python（服务端）

from wmpp_server_sdk import SMSetup
sm = SMSetup()
sm.set_server_url("http://localhost:8080")
sm.set_app_key("systemC")
sm.set_secret_key("systemC-secret")
sm.set_message("你关注的创作者发布新视频")
sm.set_target_type("USER")
sm.set_target_user_id("5001")
sm.push()
